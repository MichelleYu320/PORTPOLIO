BreatheLog

프리다이빙 기록 관리 및 팀 매칭 서비스

홈 기본정보 AI 콘텐츠 자동화 콘텐츠 플랫폼 에듀테크 글로벌 플랫폼 사이드 프로젝트

프로젝트 개요

개인 취미인 프리다이빙을 위한 기록 관리 및 팀 매칭 서비스입니다. 프리다이버와 코치의 권한을 분리하고, 체계적인 기록 관리와 팀 생성 프로세스를 통해 안전하고 효율적인 프리다이빙 활동을 지원합니다.

개발 기간

2024.01 ~ 진행중

역할

Full-stack 개발

주요 기술

Next.js, Supabase

주요 기능

로그인/회원가입 및 인증 기능

Supabase Auth를 활용한 안전한 사용자 인증 시스템을 구현했습니다.

  • 이메일 기반 회원가입 및 로그인
  • 사용자 프로필 관리 기능
  • 보안 강화된 인증 프로세스

프리다이버/코치 권한 분리

사용자 역할에 따른 차별화된 기능과 팀 생성/승인 시스템을 구축했습니다.

  • 역할 기반 접근 제어 (RBAC)
  • 팀 생성 및 멤버 승인 프로세스
  • 코치 인증 및 자격 검증

기록 관리 및 대시보드

개인 기록 추적과 성과 분석을 위한 종합 대시보드를 제공합니다.

  • 다이빙 기록 입력 및 관리
  • 개인 성과 시각화 차트
  • 목표 설정 및 진행상황 추적

외부 대회정보 API 연동

실시간 대회 정보를 제공하여 사용자들의 참여 기회를 확대합니다.

  • 외부 API 통합 및 데이터 동기화
  • 대회 일정 및 세부정보 제공
  • 참가 신청 및 알림 기능

Backend

  • Figma (디자인)

개발 과정 및 도전

사용자 경험 설계

프리다이빙 특성상 안전이 최우선이므로, 직관적이고 신뢰할 수 있는 UI/UX를 설계했습니다. 특히 기록 입력과 팀 관리 기능에서 사용자의 편의성을 최대한 고려했습니다.

실시간 데이터 동기화

Supabase의 실시간 기능을 활용하여 팀 멤버 간의 활동 상황을 실시간으로 공유할 수 있도록 구현했습니다. 이를 통해 더 나은 협업과 안전 관리가 가능해졌습니다.

권한 관리 시스템

프리다이버와 코치의 역할을 명확히 구분하고, 각각에 적합한 기능을 제공하는 권한 시스템을 구축했습니다. 이는 서비스의 전문성과 신뢰성을 높이는 핵심 요소였습니다.

풀스택 개발 경험

기획부터 디자인, 개발, 배포까지 전체 프로세스를 혼자 진행하면서 제품 개발의 전반적인 이해도를 크게 향상시킬 수 있었습니다.

최신 기술 활용

Next.js 14와 Supabase 등 최신 기술 스택을 활용하면서 빠른 개발과 안정적인 서비스 제공 간의 균형을 찾는 방법을 배웠습니다.
