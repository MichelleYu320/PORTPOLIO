선택된 문제

[문제] 함수 f(x) = x² - 4x + 3의 최솟값을 구하시오.

생성 결과

📝 해설:
이 문제는 이차함수의 최솟값을 구하는 문제입니다.

Step 1: 주어진 함수를 표준형으로 변환합니다.
f(x) = x² - 4x + 3
f(x) = (x² - 4x + 4) - 4 + 3
f(x) = (x - 2)² - 1

Step 2: 표준형에서 최솟값을 찾습니다.
(x - 2)² ≥ 0 이므로, (x - 2)² - 1 ≥ -1
등호는 x = 2일 때 성립합니다.

답: 최솟값은 -1이며, x = 2일 때 나타납니다.
