## 🚀 사용 방법

### 최신 버전 확인
```
v2_final/portfolio_integrated_final_v2.html
```
이 파일이 현재 최신 통합 버전입니다.

### 이미지 경로 참조
모든 이미지는 `images/` 폴더에 있으므로, HTML에서 참조 시:
```html
<img src="../images/파일명.png">
```
