프로젝트를 통한 인사이트

글로벌 네트워크 최적화 리딩

8개국 서비스 제공 과정에서 발생한 네트워크 품질 이슈를 해결하기 위해 AWS 팀과 직접 협업했습니다. 여러 리전의 서버 구성과 CDN 최적화, 네트워크 라우팅 테스트를 주도하며 각 국가별 최적의 인프라 아키텍처를 설계했고, 이를 통해 전체 서비스 지연시간을 평균 40% 단축시켰습니다.

기술팀과의 협업 리더십

AWS 엔지니어들과 함께 서버 성능 테스트를 진행하고, WebRTC 프로토콜 최적화를 위한 기술적 의사결정을 리딩했습니다. PM으로서 비즈니스 요구사항과 기술적 제약 사이에서 균형점을 찾고, 글로벌 서비스 안정성을 확보하는 방법을 체득했습니다.

VoC 기반 지속적 개선

사용자 피드백을 체계적으로 수집하고 분석하여 2회의 대규모 리뉴얼을 성공적으로 진행했습니다. 데이터 기반 의사결정과 빠른 실험을 통해 제품을 지속적으로 개선하는 프로세스를 구축했습니다.
